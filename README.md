#  AquaDash UI - Static Dashboard

A modern and clean administrative dashboard interface built with **HTML5** and **CSS3**. This project focuses on UI design and layout structure.

##  Features
* **Overview Stats:** Visual cards showing daily visits, new users, sales, and support tickets.
* **Widget Layout:** A dedicated space for monitoring tasks and new widgets.
* **Transaction Table:** A structured table displaying recent project activities, hours, and progress bars.
* **Responsive Design:** Built using **Bootstrap 5** to ensure the layout looks great on different screen sizes.

##Tech Stack
* **HTML5** 
* **CSS3** 
* **Bootstrap 5:** Used for the Grid system and responsive utility classes.
